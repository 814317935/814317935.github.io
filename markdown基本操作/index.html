Markdown语法总结

Veröffentlicht am 2018-10-03 | Edited on 2018-10-26

本文源自github-Markdown教程

Markdown是一种轻量级且易于使用的语法，其实质上是一种在网络上设置文字样式的方法。形式为带 .md 或 .markdown 扩展名的文件

用途：控制文档的显示；将单词格式化为粗体或斜体；添加图像；创建列表 …

语法：

编辑文本

粗体 也是粗体: **粗体** __也是粗体__
斜体 还是斜体: *斜体* _还是斜体_
百度: [百度](https://www.baidu.com)

编辑列表

  1. 内容一: 1. 内容一
  2. 内容二: 1. 内容二
  3. 内容三: 1. 内容三
    (注意：符号1.要与内容用隔开)
  • 用字符开头: ` 用字符*开头: `
  • 形成·列表: * 形成·列表:
    (注意：符号* -要与内容用隔开)
  • 用字符-开头: - 用字符*开头:
  • 与字符作用相同: `- 与字符作用相同:`
    • 还可以形成二级列表
    • Tab加-或者*

嵌入图片

如果要嵌入图像，请按以下步骤操作：

写入代码

如果你有内联代码块，请用反引号包装它们：`var example = true`: var example = true

附加功能

GitHub支持Markdown中的许多附加功能，可以帮助您引用和链接到人。如果你想向某人发表评论，你可以在他们的名字前加上@符号：@用户名(未验证)
